HTML 4

December 18, 1997

HTML 4.0 was published as a W3C Recommendation. It offers three variations

Strict, in which deprecated elements are forbidden

Transitional, in which deprecated elements are allowed

Frameset, in which mostly only frame related elements are allowed

April 24, 1998

HTML 4.0 was reissued with minor edits without incrementing the version number

December 24, 1999

HTML 4.01 was published as a W3C Recommendation. It offers the same three variations as HTML 4.0 and its last errata were published on May 12, 2001

May 2000

ISO/IEC 15445:2000 ("ISO HTML", based on HTML 4.01 Strict) was published as an ISO/IEC international

standard. In the ISO this standard falls in the domain of the ISO/IEC JTC1/SC34 (ISO/IEC Joint Technical

Committee 1, Subcommittee 34 – Document description and processing languages)

After HTML 4.01, there was no new version of HTML for many years as development of the parallel, XML-based

language XHTML occupied the W3C's HTML Working Group through the early and mid-2000s.

What's new in HTML 5

Semantics

allowing you to describe more precisely what your content is.

<nav>

defines a section that contains navigation links that appear often on a site.

<article>

defines a piece of self-contained content. It does not refer to the main content alone and can be used for comments and widgets.

<section>

defines a section of a document to indicate a related grouping of semantic meaning.

<header>

defines a page which typically contains the logo, title, and navigation.
